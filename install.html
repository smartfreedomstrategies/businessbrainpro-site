Install & setup

Getting your Business Brain in.

You'll get your plugin file from me directly — it comes with your license, not from a download page. Setup is something we do together over your first session. This page is the reference: how the file goes in, how to wake it up, and the quick fixes for the usual snags.

Install it

  1. Get your file

    I send you your Business Brain — a single .plugin file. Save it somewhere you can find it (your Downloads folder is fine).

  2. Add it in the Claude desktop app

    Open the Claude desktop app and add the file as a personal plugin.

    Customize → Personal plugins → + → Create plugin → Upload plugin → pick the file I sent
  3. Start fresh and say the word

    Plugins load when a chat begins, so open a new chat and say set up my command center. That's the moment your Business Brain wakes up and starts building your system with you.

Something not working?

  • Not all your tools showed up

    Usually an older file or an app that needs a restart. Ask me for the latest file, re-upload it (step 2), then fully quit and reopen the desktop app — plugins only reload on a clean start.

  • The file won't upload

    Make sure you're on the latest Claude desktop app and uploading the .plugin file exactly as I sent it (not unzipped, not renamed). If it still refuses, send me a screenshot and I'll sort it.

  • The app won't open, or it's running slow

    That's before the Business Brain is even involved. Restart the app — and your computer if it's sluggish — update to the latest version, then try step 2 again.

Stuck on any of this? Email me and I'll walk you through it. This is done with you — never left to a manual.
